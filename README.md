## LSP - Local Salesforce Package

LSP is CRM, it was made on top of Salesforce platform to run YWAM bases.

#### Requirements

We assume that you have [Communities User Licenses](https://help.salesforce.com/articleView?id=users_license_types_communities.htm&language=en_US&type=0) activated on your Salesforce instance (org).
